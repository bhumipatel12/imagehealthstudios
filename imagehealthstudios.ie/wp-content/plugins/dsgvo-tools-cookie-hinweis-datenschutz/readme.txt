Adds a full customizable bar with cookie notice and integrated notice to privacy policy

== Description ==
As reqired in the laws of GDPR, websites has to notify the user about usage of cookies. Also the data privacy has to be mentioned.
Exactly this is the function of the plugin: It adds an full customizable bar with cookie notice and hyperlink to privacy policy page.
However, it doesn't block cookies. The plugin is full customizable and GDPR compliant.
The GDPR conformity hasn't been proved by an lawyer and only reflects the estimation of the author. If you have further questions legal advice is required.

Compatible with Polylang, supports a privacy page for each language!

== Installation ==
1. Click at 'install'
2. Activate the plugin through the plugin menu in WordPress
3. Customize the plugin at backend menu 'GDPR tools: cookies + privacy'
4. Done!
